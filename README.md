# 2D-игра на Unity

Учебный проект простой 2D-игры, созданный на игровом движке Unity. Игра подходит для начинающих разработчиков и демонстрирует основы работы с Unity и C#.

## Описание

В этой игре игрок управляет персонажем, который должен собирать бонусы и избегать препятствий. Проект включает базовые механики 2D-игр: 
- Движение персонажа
- Сбор очков
- Система жизней/попыток
- Фоновая музыка и простые эффекты

## Технологии

- Unity (версия X.X.X, укажите свою)
- C#
- 2D Sprites
