# 2D-игра на Unity

Учебный проект простой 2D-игры, созданный на игровом движке Unity. Игра подходит для начинающих разработчиков и демонстрирует основы работы с Unity и C#.

## Описание

В этой игре игрок управляет процессом развития башни, которая побеждает врагов. Проект включает базовые механики 2D-игр: 
- Движение персонажей
- Сбор очков
- Система жизней
- Система прокачки

## Технологии

- Unity 
- C#
- 2D Sprites
